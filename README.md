# I CAN – AI English Learning App

## Overview

I CAN is a mobile-first English learning app that uses AI to generate personalized learning content tailored to each user’s level.

The core idea is simple:

> Learners improve fastest when content is slightly above their current level.

I CAN continuously adapts to the learner by generating stories, audio, and exercises that are approximately 3% more difficult than their current ability.

---

## Core Value Proposition

* Personalized learning content generated in real-time
* Focus on vocabulary acquisition and comprehension
* Integrated listening and speaking practice
* Immediate feedback on pronunciation and weak areas

---

## Target Users

* Chinese speakers learning English
* Beginner to intermediate learners
* Users who prefer short, daily learning sessions (5–15 minutes)
* Mobile-first users who want a simple, engaging experience

---

## Core Features (V1 Scope)

1. Adaptive story generation based on user level
2. Highlighted new vocabulary within stories
3. Audio playback for stories (TTS)
4. Shadowing modes (phrase-by-phrase, sentence-level)
5. Basic vocabulary flashcards (word + audio)
6. Simple progress tracking (daily usage, streak)

---

## Future Features (Post-V1)

* Pronunciation analysis and feedback
* AI-generated speaking drills
* Personalized weakness tracking
* Topic-based learning paths
* Social or competitive features

---

## Product Philosophy

* Keep sessions short and engaging
* Prioritize comprehension over memorization
* Use AI to dynamically adapt difficulty
* Avoid overwhelming the user with too many features

---

## Monetization Strategy

* Freemium model

  * Free: limited daily usage (e.g., 1 story per day)
  * Paid: expanded usage, full audio features, advanced feedback

Potential future:

* Credit-based system for heavy AI usage (story generation, pronunciation analysis)

---

## Platform

* Mobile app (iOS first)
* Built using React Native (Expo)

---

## Technical Overview (High-Level)

* Frontend: React Native (Expo)
* Backend: lightweight API (Node.js or Python)
* AI services:

  * Text generation (stories)
  * Text-to-speech (audio)
  * Speech-to-text (pronunciation feedback)

---

## Development Approach

* Start with a focused MVP
* Build core loop first: generate → read → listen → repeat
* Iterate quickly using Claude Code
* Avoid over-engineering early

---

## Vision

To become the most effective and engaging way for learners to improve their English through personalized, AI-driven content.
